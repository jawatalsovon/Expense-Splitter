Tracking shared expenses manually is a mess.

Who paid for dinner? Who owes whom for groceries? Did Sajin pay me back yet?

When you're living with flatmates, travelling with friends, or splitting costs within any group — these small questions pile up fast and get awkward even faster.

So I built an app to solve exactly that.

Hisab is a real-time expense splitting web app that helps any group track shared costs and settle debts without the confusion.

🔹 Create a group and invite members via a code
🔹 Log expenses with flexible payer splits — one person paid full, or multiple people paid different amounts
🔹 Choose exactly who shares each expense — not always the whole group
🔹 See simplified settlements: the minimum transactions needed to zero out all debts
🔹 Record repayments when someone pays back — balances update instantly
🔹 Full real-time sync — everyone sees updates the moment they happen

Use cases go beyond just flatmates:
📌 Friends trip — hotel, transport, meals, activities
📌 Group project expenses
📌 Family gatherings
📌 Office lunch pools
📌 Any situation where a group shares costs unevenly

Built with React, Supabase, and Tailwind CSS. Supports both English and Bangla, with light and dark mode.

Would love feedback from anyone who has dealt with this problem.

🔗 App link: https://hisab-expense-splitter.vercel.app/
